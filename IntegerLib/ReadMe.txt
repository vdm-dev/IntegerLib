========================================================================
    CONSOLE APPLICATION : IntegerLib Project Overview
========================================================================

IntegerLib.vcproj
    This is the main project file for VC++ projects generated using an Application Wizard. 
    It contains information about the version of Visual C++ that generated the file, and 
    information about the platforms, configurations, and project features selected with the
    Application Wizard.

test.c
    This is the main application source file.
    
add.c
	This file contains the addition functions
	
div.c
	This file contains the division functions
	
conversions.c
	This file contains the conversion functions as well as the 
	error handling functions
	
mult.c
	This file contains the multiplication functions
	
shift.c
	This file contains the shift functions
	
sub.c
	This file contains the subtraction functions
	
unary.c
	This file contains the unary functions

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named IntegerLib.pch and a precompiled types file named StdAfx.obj.
